JavaScript Fundamentals

A practical demonstration of variables, operators, and control statements

Variables

// Declaring variables
var message = "Hello, World!";
let count = 5;
const MAX_VALUE = 100;

// Outputting variables
console.log(message);
console.log("Count: " + count);
console.log("Max value: " + MAX_VALUE);
Output will appear here

Variables are containers for storing data values. In JavaScript, we can use var, let, or const to declare variables.

const is used for values that won't change, let for variables that may be reassigned, and var is the older way to declare variables.

Operators

// Arithmetic operators
let x = 10;
let y = 3;

let sum = x + y;
let difference = x - y;
let product = x * y;
let quotient = x / y;
let remainder = x % y;

// Comparison operators
let isEqual = x === y;
let isGreater = x > y;

// Output results
console.log("Sum: " + sum);
console.log("Is equal: " + isEqual);
Output will appear here

JavaScript operators are used to assign values, compare values, perform arithmetic operations, and more.

Arithmetic operators: +, -, *, /, %
Comparison operators: ==, ===, !=, !==, >, <, >=, <=

If Statement

let temperature = 25;

if (temperature > 30) {
  console.log("It's hot outside!");
} else if (temperature > 20) {
  console.log("The weather is pleasant.");
} else {
  console.log("It's cold outside!");
}

console.log("Temperature is: " + temperature);
Output will appear here

Conditional statements are used to perform different actions based on different conditions.

Use if to specify a block of code to be executed if a condition is true.

Use else to specify a block of code to be executed if the condition is false.

Use else if to specify a new condition to test if the first condition is false.

Switch Statement

let day = new Date().getDay();
let dayName;

switch (day) {
  case 0:
    dayName = "Sunday";
    break;
  case 1:
    dayName = "Monday";
    break;
  case 2:
    dayName = "Tuesday";
    break;
  case 3:
    dayName = "Wednesday";
    break;
  case 4:
    dayName = "Thursday";
    break;
  case 5:
    dayName = "Friday";
    break;
  case 6:
    dayName = "Saturday";
    break;
  default:
    dayName = "Unknown";
}

console.log("Today is: " + dayName);
Output will appear here

The switch statement is used to perform different actions based on different conditions.

It's often used as an alternative to an if-else if-else chain when you have many conditions to check.

The break keyword breaks out of the switch block and stops the execution of more code inside the switch.

For Loop

// Simple for loop example
console.log("Counting from 1 to 5:");

for (let i = 1; i <= 5; i++) {
  console.log("Count: " + i);
}

// Loop through an array
const fruits = ["Apple", "Banana", "Orange", "Mango"];
console.log("Fruits array:");

for (let i = 0; i < fruits.length; i++) {
  console.log("Fruit " + i + ": " + fruits[i]);
}
Output will appear here

Loops can execute a block of code a number of times. The for loop is the most common type of loop.

Syntax: for (initialization; condition; increment) { code }

Loops are handy if you want to run the same code over and over again, each time with a different value.

While Loop

// Simple while loop example
console.log("Counting down from 5:");

let count = 5;
while (count > 0) {
  console.log("Countdown: " + count);
  count--;
}

console.log("Liftoff!");

// Do-while example
console.log("Do-while example:");
let i = 1;
do {
  console.log("Number: " + i);
  i++;
} while (i <= 3);
Output will appear here

The while loop loops through a block of code as long as a specified condition is true.

The do-while loop is a variant of the while loop that will execute the code block once, before checking if the condition is true.

Be careful with while loops - they can run infinitely if the condition never becomes false!
